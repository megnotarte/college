# College Chartered

Marketing site for College Chartered, a college application management service.

Live at [megnotarte.github.io/college](https://megnotarte.github.io/college)

## Stack

Vanilla HTML, CSS, and JavaScript — no build step, no dependencies.

## Development

Open `index.html` directly in a browser, or serve it locally:

```bash
npx serve .
```

## Deployment

Deployed via GitHub Pages from the `main` branch. Pushing to `main` publishes automatically.
